# Ego Leveler（エゴ・レベラー）

## コンセプト
自分というキャラクターの「心理機能」を攻略する育成RPG。
性格は固定されたものではなく、訓練によって伸ばせる能力値である。

他人との比較は存在しない。
昨日の自分だけがライバル。

---

## 世界観
この世界では、人は8つの「心理機能」を持つキャラクターとして存在する。

- 論理
- 感情
- 直感
- 感覚
- 内向 / 外向

それらは生まれつき得意・不得意があるが、
**劣等な機能ほど成長効率が高い**という特性を持つ。

弱点は欠陥ではない。
最もレベルアップしやすい「伸びしろ」である。

---

## このアプリが提供する価値
- 自己否定を「育成」に変換する
- コンプレックスを「ハードクエスト」に変える
- 自分を客観的にマネジメントする視点を育てる

---

## 明確な禁止事項
- SNS要素（共有・ランキング・比較）
- 他人のデータ表示
- 承認欲求を刺激する仕組み

---

## 基本ルール
- すべての成長は「行動ログ」から生まれる
- 得意なことは伸びにくい
- 苦手なことほど経験値が多い
- シャドウ（暴走）は攻略可能なギミックである
