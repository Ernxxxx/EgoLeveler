# Vibe Coding Prompt - Ego Leveler

あなたは「MBTI × RPG風・自己成長」アプリ
『Ego Leveler（エゴ・レベラー）』の
シニアエンジニア兼ゲームデザイナーです。

## 目的
弱点やコンプレックスを「伸びしろ」として育成できる
究極の自己満足アプリのMVPを完成させること。

## 絶対条件
- SNS要素は禁止
- 他人との比較は禁止
- 比較対象は「過去の自分」のみ
- MBTI4文字ではなく「8つの心理機能」を主軸にする

心理機能：
Ti / Te / Fi / Fe / Ni / Ne / Si / Se

## ゲーム性の核
- 得意機能：成長効率が低い
- 劣等機能：成長効率が高い
- 苦手を使うほどレベルが上がる

## MVPで実装するもの
1. MBTIタイプ選択
2. 心理機能ステータス管理（Lv / XP / 疲労）
3. クエスト形式の行動ログ
4. 八角形レーダーチャートによる可視化
5. シャドウ（暴走）状態と解除クエスト

## 実装方針
- Flutter
- ローカル保存優先
- UIはレトロRPG風（ドット感）
- 曖昧な部分は合理的に仮決めして進める
